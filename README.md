# MyLetterBox 

**MyLetterBox** — это консольное C++ приложение для управления вашей библиотекой фильмов и сериалов, вдохновленное популярным сервисом Letterboxd. Проект позволяет просматривать базу данных медиа, искать проекты по жанрам и ключевым словам, а также вести собственный профиль с избранным, просмотренным и отзывами.

Проект написан с использованием современных стандартов C++ (C++23) и демонстрирует применение принципов ООП, работу с файловой системой, умными указателями и алгоритмами стандартной библиотеки.

## Основной функционал

*   **База данных:** Загрузка списка фильмов и сериалов из `.csv` файлов.
*   **Продвинутый поиск:** Фильтрация по точному названию, жанру или ключевым словам.
*   **Система профиля:** Создание пользователя при входе в систему.
*   **Управление списками:** Возможность добавлять медиа в списки «Избранное» (Favorites) и «Просмотренное» (Watched).
*   **Отзывы и оценки:** Пользователи могут оставлять рецензии на фильмы/сериалы с оценкой от 1 до 10.
*   **Аналитика:** Автоматический подсчет общего времени, потраченного на просмотр всех тайтлов в списке «Просмотренное».

## Сборка и запуск проекта

### Для CLion:
1. В верхней панели (рядом с кнопкой Run) нажмите на выпадающий список и выберите **Edit Configurations...**
2. Слева выберите ваш профиль запуска.
3. Найдите поле **Working directory**.
4. Укажите в этом поле **абсолютный путь до корневой папки вашего проекта**  .
5. Нажмите `Apply` и потом`OK`.

## Запуск и использование

После правильной настройки директории, скомпилируйте и запустите проект. При старте программа сообщит, сколько фильмов и сериалов было успешно загружено.
